# Segment 05

**Original (ZH)**
釋迦牟尼的佛教本質上就是人間的佛教。佛陀降誕世間，其教化皆為了解決人間問題，增進人類幸福安樂。雖然「佛教就是人間佛教」的觀點尚未得到佛教界及學術界的普遍一致承認，但星雲大師的這一說法清晰揭示了佛陀教導的精神本質以及早期佛教與現代人間佛教之間的內在關聯。

**Translation (JA)**
釈迦牟尼の仏教は本質的に人間のための仏教である。仏陀は人間の問題を解決し人類の幸福と安楽を増進するために世に出現した。「仏教とはすなわち人間仏教である」という見解は仏教界や学界でまだ普遍的な合意を得ていないものの、星雲大師のこの主張は仏陀の教えの精神的本質と、初期仏教と現代人間仏教の内在的な連関を明確に示している。
